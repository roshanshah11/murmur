# Code of conduct

Murmur adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) by reference. The canonical text governs in case of any conflict with the summary below.

This project is committed to a respectful, welcoming environment for everyone who participates — contributors, users, reviewers, and maintainers. Harassment, demeaning conduct, and personal attacks are not tolerated.

Disagreements about technical direction are expected and healthy. Disagreements about how people are treated are not negotiable.

## Reporting

If you experience or witness behavior that violates this code, contact the maintainer directly at `ashah@alixpartners.com`. Reports are handled confidentially. Include what happened, when, where, and any links or context that would help an outside reader understand the situation.

The maintainer will acknowledge the report within 72 hours and follow up with next steps. Enforcement actions can include warning, temporary ban, or permanent ban from project spaces, applied at the maintainer's discretion and consistent with the Contributor Covenant's enforcement guidelines.

## Scope

This code applies to all project spaces: the GitHub repository, issue tracker, pull requests, Discussions, and any public communication where someone is representing Murmur.
