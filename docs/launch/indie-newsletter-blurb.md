# Murmur

Double-tap fn, speak, paste — Murmur is a new macOS dictation app that runs whisper.cpp on-device and never touches the network during transcription. It's the local-first answer to apps like Wispr Flow: a little less accurate on noisy audio, a lot more comfortable if you'd rather your microphone stay private. Free, MIT-licensed, macOS 13+ on both Apple Silicon and Intel. v1.0 ships with editable vocabulary, four cleanup profiles (Raw, Casual, Formal, Code), a whisper model picker from tiny through large-v3, an opt-in history viewer, and Sparkle in-app updates. No account, no telemetry, no subscription.

`brew install --cask murmur` — or grab the DMG at murmur-landing-phi.vercel.app. Source at github.com/roshanshah11/murmur.
