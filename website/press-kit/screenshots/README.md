# Murmur — Screenshots

Six canonical shots that tell the whole story. All shots should be taken on a 14" or 13" MacBook Pro running the v1.0 build, in light appearance, with the app's paper background visible.

Capture at native resolution (Retina). The 14" MBP gives `3360 × 2100`; the 13" gives `2880 × 1864`. Either is acceptable; the 14" is preferred because the notch overlay reads at a comfortable size.

Status: **owed — capture from running v1.0 build.**

## How to capture

1. Set the system appearance to Light.
2. Quit anything noisy; clean menu bar (`Bartender` or `hidden`).
3. Pick a wallpaper that is solid or near-solid in a warm neutral tone.
4. Use `Cmd+Shift+4`, then `Space`, then click the Murmur window — gives a clean shadow with no clutter.
5. Save as PNG. Do not compress.
